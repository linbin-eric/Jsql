概述

JSql是一个高性能的Java SQL映射框架，旨在简化数据库操作。它通过接口绑定和流式API提供透明的数据库访问，支持动态SQL、自动结果映射和多种数据库方言。

核心特性包括：

  • 基于接口的SQL绑定
  • 流式Model API构建SQL查询
  • 支持动态SQL
  • 自动对象映射
  • 实体注解映射
  • 多数据库支持

核心组件

SessionFactory

SessionFactory是JSql的入口点，用于创建SqlSession实例。它管理着数据库连接池和框架配置。

// 创建SessionFactory的典型方式
SessionFactoryConfig config = new SessionFactoryConfig();
config.setDataSource(dataSource);
// 可以添加其他配置，如扫描包、SQL执行器等
SessionFactory sessionFactory = config.build();

SqlSession

SqlSession代表一个数据库会话，提供CRUD操作、事务管理和Mapper访问。每次数据库交互都从SqlSession开始。

// 获取并使用SqlSession
SqlSession session = sessionFactory.openSession();
try {
    // 执行数据库操作
    User user = session.findOne(Model.select(User::getName).from(User.class).where(Param.eq(User::getId, 1)));
    // 提交事务（如果需要）
    session.commit();
} finally {
    // 确保session被关闭
    session.close();
}

Model API

Model API是JSql提供的流式API，用于以编程方式构建SQL查询。它提供了类型安全和链式调用的特性。

基本查询

// 查询所有字段
QueryModel model1 = Model.selectAll(User.class);

// 查询特定字段
QueryModel model2 = Model.select(User::getName, User::getAge).from(User.class);

// 带条件查询
QueryModel model3 = Model.selectAll(User.class)
                        .where(Param.eq(User::getAge, 18));

// 分页查询
Page page = new Page(10, 0); // 每页10条，偏移量0
QueryModel model4 = Model.selectAll(User.class)
                        .where(Param.gt(User::getAge, 18))
                        .page(page);

插入操作

// 插入特定字段
InsertModel insertModel = Model.insert(User.class)
                               .insert(User::getName, "张三")
                               .insert(User::getAge, 25);

// 插入实体对象
User user = new User();
user.setName("李四");
user.setAge(30);
InsertEntityModel insertEntityModel = Model.insert(user);

更新操作

// 更新特定字段
UpdateModel updateModel = Model.update(User.class)
                               .set(User::getAge, 26)
                               .where(Param.eq(User::getName, "张三"));

删除操作

// 删除记录
DeleteModel deleteModel = Model.deleteFrom(User.class)
                               .where(Param.eq(User::getName, "张三"));

Mapper API

Mapper API通过Java接口将方法与SQL语句绑定，实现自动代理生成。这是JSql推荐的数据库操作方式。

定义Mapper接口

import com.jfirer.jsql.annotation.Mapper;
import com.jfirer.jsql.annotation.Sql;
import com.jfirer.jsql.model.Param;
import java.util.List;

@Mapper(User.class) // 声明该Mapper关联的实体类
public interface UserMapper {
    // 使用@Sql注解定义SQL语句
    @Sql("select * from user where name = ${name}")
    User findByName(String name);
    
    @Sql("select * from user where age > ${age}")
    List<User> findByMinAge(int age);
    
    @Sql("insert into user(name, age) values(${user.name}, ${user.age})")
    int insertUser(User user);
    
    @Sql("update user set age = ${age} where name = ${name}")
    int updateAgeByName(String name, int age);
    
    @Sql("delete from user where name = ${name}")
    int deleteByName(String name);
}

使用Mapper

// 从SqlSession获取Mapper实例
UserMapper userMapper = session.getMapper(UserMapper.class);

// 调用Mapper方法
User user = userMapper.findByName("张三");
List<User> users = userMapper.findByMinAge(18);
int result = userMapper.insertUser(new User("王五", 28));

注解

JSql使用注解来配置实体类与数据库表的映射关系。

@TableDef

将Java类映射到数据库表。

@TableDef("user")
public class User {
    // ...
}

@Pk

标记字段为主键。

public class User {
    @Pk
    private Integer id;
    // ...
}

@ColumnName

将字段映射到指定的数据库列。

public class User {
    @ColumnName("user_name")
    private String name;
    // ...
}

@AutoIncrement

标记主键为自增字段。

public class User {
    @Pk
    @AutoIncrement
    private Integer id;
    // ...
}

@SqlIgnore

标记字段不参与SQL操作。

public class User {
    @SqlIgnore
    private String temporaryField;
    // ...
}

使用示例

完整示例：用户管理

// 1. 定义实体类
@TableDef("user")
public class User {
    @Pk
    @AutoIncrement
    private Integer id;
    
    @ColumnName("user_name")
    private String name;
    
    private int age;
    
    // 构造函数、Getter和Setter省略
}

// 2. 定义Mapper接口
@Mapper(User.class)
public interface UserMapper {
    @Sql("select * from user where id = ${id}")
    User findById(int id);
    
    @Sql("select * from user")
    List<User> findAll();
    
    @Sql("insert into user(user_name, age) values(${user.name}, ${user.age})")
    int insert(User user);
    
    @Sql("update user set user_name = ${user.name}, age = ${user.age} where id = ${user.id}")
    int update(User user);
    
    @Sql("delete from user where id = ${id}")
    int deleteById(int id);
}

// 3. 使用示例
public class UserService {
    private SessionFactory sessionFactory;
    
    public UserService(SessionFactory sessionFactory) {
        this.sessionFactory = sessionFactory;
    }
    
    public User getUser(int id) {
        try (SqlSession session = sessionFactory.openSession()) {
            UserMapper mapper = session.getMapper(UserMapper.class);
            return mapper.findById(id);
        }
    }
    
    public void addUser(User user) {
        try (SqlSession session = sessionFactory.openSession()) {
            UserMapper mapper = session.getMapper(UserMapper.class);
            mapper.insert(user);
            session.commit();
        }
    }
}
